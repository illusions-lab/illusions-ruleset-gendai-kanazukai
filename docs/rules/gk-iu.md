# gk-iu — 動詞「言う」の正書法（「いう」と書く）

- **ruleId**: `gk-iu`
- **ガイドライン**: 現代仮名遣い (`gendai-kanazukai-1986`)
- **レベル**: L2（形態素解析）
- **既定**: 有効 / severity = warning
- **適用モード**: 小説 / 公用文 / ブログ / 学術 / SNS（全モードで自動有効）

## 意図

現代仮名遣い（昭和61年内閣告示第1号）本文 第2の2 では、語の現代語の発音に基づく
仮名遣いを定めるが、動詞「言う」は歴史的仮名遣い「いふ」の「い」音を保ち「いう」と書く。
口語の発音では「ゆう（ゆった・ゆって）」となることが多く、この発音につられて誤記されやすい。

本ルールは形態素解析トークンを利用して、動詞「言う」が「ゆう」系の活用形で記されている
箇所を検出し、「いう」系への修正を提案する。

対象活用形の例（「ゆう」系 → 「いう」系）:
- ゆう → いう
- ゆった → いった
- ゆって → いって
- ゆわない → いわない
- ゆえば → いえば

## 出典

現代仮名遣い（昭和61年内閣告示第1号）本文 第2の2

## 例

| 区分 | テキスト                    | 期待                   |
| ---- | --------------------------- | ---------------------- |
| 正例 | `先生がいったことを思い出した。` | 指摘なし               |
| 誤例 | `先生がゆったことを思い出した。` | `ゆっ`→`いっ` に修正   |
| 誤例 | `そうゆうことだ。`            | `ゆう`→`いう` に修正   |
| 正例 | `髪をゆう。`                | 「結う」のため指摘なし  |

## 偽陽性回避

- POS が「動詞」のトークンのみ対象とする。名詞「夕（ゆう）」・形容詞・副詞は除外。
- 判定基準 A（優先）: `basic_form === "言う"` が確認できた場合のみ検出。
  kuromoji は「ゆう」を言う の口語活用と認識し `basic_form="言う"` を返すことが多い。
- 判定基準 B（fallback）: `basic_form` が不明・`"*"` の場合、`reading` が「ユ」で始まり
  かつ `surface` が `IU_SURFACE_MAP`（ゆう/ゆわ/ゆい/ゆっ/ゆえ）に含まれる場合のみ検出。
  wide match（先頭が「ゆ」であれば何でも）は行わない。
- 動詞「結う」は `basic_form="結う"` となるため基準 A によって除外される。
  基準 B の場合も `IU_SURFACE_MAP` は極小なため「結う」の「ゆい」等が誤爆しにくいが、
  読み（ユイ/ユウ）は言う同様のため、`basic_form` が取れない文脈では注意が必要。

## 実装メモ

- `ctx.bases.AbstractMorphologicalLintRule` を継承し、`lintWithTokens()` で実装する。
- `lint()` は空配列を返す（L2 ルールの標準パターン）。
- `deriveCorrection(surface)` が surface の先頭「ゆ」を「い」に置換して修正形を返す。
  `IU_SURFACE_MAP` で完全一致する場合は優先して返す。
