# gk-yotsugana — 四つ仮名（ぢ・づ）の使い分け

- **ruleId**: `gk-yotsugana`
- **ガイドライン**: 現代仮名遣い (`gendai-kanazukai-1986`)
- **レベル**: L1（正規表現）
- **既定**: 有効 / severity = warning
- **適用モード**: 小説 / 公用文 / ブログ / 学術 / SNS（全モードで自動有効）

## 意図

現代仮名遣い（昭和61年内閣告示第1号）本文 第2の5(1)(2) は、次の語に「ぢ」「づ」を用いると定める。

- **(1) 同音の連呼**: ちぢみ・ちぢむ・ちぢれる・ちぢこまる／つづみ・つづら・つづく・つづめる・つづる
- **(2) 二語の連合（連濁）**: はなぢ（鼻血）・そこぢから（底力）・ちかぢか（近々）・ちりぢり・みかづき（三日月）・たけづつ（竹筒）・たづな（手綱）・こづつみ（小包）・こづかい（小遣）・てづくり（手作）・みちづれ（道連）・かたづく・もとづく・うらづける・ゆきづまる・つくづく・つねづね（常々）・つれづれ など

これらが「じ」「ず」と誤記された箇所を検出し、「ぢ」「づ」への修正を提案する。収録語は告示の語例に基づく
（連濁の語例は告示に多数あり、本ルールは偽陽性が起きにくい語から順次収録する）。

## 出典

現代仮名遣い（昭和61年内閣告示第1号）本文 第2の5(1)

## 例

| 区分 | テキスト         | 期待                  |
| ---- | ---------------- | --------------------- |
| 正例 | `シャツがちぢむ。` | 指摘なし              |
| 誤例 | `シャツがちじむ。` | `ちじ` を `ちぢ` に修正 |
| 誤例 | `みかずきが見える。` | `みかず` を `みかづ` に修正 |
| 誤例 | `はなじが出た。` | `はなじ` を `はなぢ` に修正 |

## 偽陽性回避

- 各語は「誤記された二字＋直後の語形（先読み）」でのみ一致させ、別語への巻き込みを防ぐ。
  - `知事(ちじ)` `図画(ずが)` `地面(じめん)` は直後が対象語形でないため不一致。
- 告示 第2-5(2) の**許容語**（本則「じ・ず」、「ぢ・づ」も可）は誤りではないので**指摘しない**。
  - `いなずま(稲妻)` `せかいじゅう(世界中)` `うなずく` `つまずく` `ひとりずつ` `さかずき(杯)` `ほおずき`
    `ゆうずう(融通)` `きずな(絆)` など。
- 告示 第2-5(1) の**〔注意〕**（この例にあたらない＝じ）も対象外: `いちじく` `いちじるしい`。
- 告示 第2-5(2) の**〔注意〕**（漢字音でもともと濁る＝じ・ず）も対象外: `じめん(地面)` `ぬのじ(布地)`
  `ずが(図画)` `りゃくず(略図)`。
- 語形の先読みで別語を除外: `はなじろむ(鼻白む)` `たずねる(訪ねる)` `ゆきずり` `連れずに` `てずから`
  `固唾(かたず＋を)` は一致しない。

## 実装メモ

- `ctx.toolkit.regexReplace` を語形ごとに適用し、先読み `(?=…)` / 否定先読み `(?!…)` で
  対象を限定。最後に `ctx.toolkit.dedupe` で同一スパンの重複を除去。
- 対象語は内閣告示の列挙に基づく**有限リスト**。語を追加する際は、誤記形が別語と衝突しないこと
  （偽陽性テスト）を必ず確認する。

## 今後の拡張候補

- **お列長音の「お」表記語**（告示 第2の6）→ `gk-o-choon` として**実装済み**。偽陽性リスクの高い
  `とおい→とうい`（統一・党員 等）・`こおり→こうり`（小売）等は意図的に除外し、誤記形が別語と
  衝突しない語のみ収録している。詳細は `docs/rules/gk-o-choon.md`。
- **連濁の追加語**（わしづかみ・こころづくし・ねばりづよい 等）も、偽陽性テストを付けつつ順次収録。
