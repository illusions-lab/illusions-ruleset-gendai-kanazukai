# gk-particle-wa-o-e — 助詞「は」「を」「へ」の正書法

- **ruleId**: `gk-particle-wa-o-e`
- **ガイドライン**: 現代仮名遣い (`gendai-kanazukai-1986`)
- **レベル**: L2（形態素解析）
- **既定**: 有効 / severity = info
- **適用モード**: 小説 / 公用文 / ブログ / 学術 / SNS（全モードで自動有効）

## 意図

現代仮名遣い（昭和61年内閣告示第1号）本文 第2の1 は、次のように定める。

> 助詞「は」「を」「へ」は、それぞれ「は」「を」「へ」と書く。

口語やかな入力の誤りで「は」→「わ」、「を」→「お」、「へ」→「え」と書かれることがある。
本ルールは形態素解析トークンを利用して、助詞として機能しているにもかかわらず誤記されている
「わ」「お」「え」を検出し、「は」「を」「へ」への修正を提案する。

## 出典

現代仮名遣い（昭和61年内閣告示第1号）本文 第2の1

## 例

| 区分 | テキスト              | 期待                    |
| ---- | --------------------- | ----------------------- |
| 正例 | `彼女は東京へ向かった。` | 指摘なし                |
| 誤例 | `彼女わ東京え向かった。` | `わ`→`は`、`え`→`へ` に修正 |
| 誤例 | `本お読む。`           | `お`(格助詞)→`を` に修正 |
| 正例 | `そうだわ。`           | 終助詞「わ」のため指摘なし |

## 偽陽性回避

- POS が「助詞」のトークンのみ対象とする。助詞以外（接頭辞の「お」、感動詞の「え」など）は対象外。
- **お → を**: 格助詞・係助詞・副助詞の「お」は「を」の誤記として安全に検出できる。
- **え → へ**: 格助詞の「え」は「へ」の誤記として安全に検出できる。
- **わ → は**: 女性語的な文末の「わ」（終助詞、例:「そうだわ」「いいわよ」）は正しい日本語であり検出しない。
  係助詞・副助詞としての「わ」（例:「わたしわ行く」）のみ「は」の誤記として検出する。
- severity を `info` に設定している（偽陽性リスクが相対的に高いため）。

## 実装メモ

- `ctx.bases.AbstractMorphologicalLintRule` を継承し、`lintWithTokens()` で実装する。
- `lint()` は空配列を返す（L2 ルールの標準パターン）。
- `わ` の検出は `pos_detail_1` が `係助詞` または `副助詞` の場合に限定する。
  kuromoji が曖昧な場合（`副助詞／並立助詞／終助詞` 等）はセットに含め対応する。
