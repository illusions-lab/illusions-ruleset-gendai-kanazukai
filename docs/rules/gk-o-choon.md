# gk-o-choon — オ列長音の例外語（「お」表記）

- **ruleId**: `gk-o-choon`
- **ガイドライン**: 現代仮名遣い (`gendai-kanazukai-1986`)
- **レベル**: L1（正規表現）
- **既定**: 有効 / severity = warning
- **適用モード**: 小説 / 公用文 / ブログ / 学術 / SNS（全モードで自動有効）

## 意図

現代仮名遣い（昭和61年内閣告示第1号）本文 第2の6 は、オ列の長音は原則としてオ列の仮名に「う」を
添えて書く（例: おとうさん・とうだい〔灯台〕）と定める。ただし次の語は、歴史的仮名遣いでオ列の仮名に
「ほ」「を」が続いたものであり、**「お」を添えて書く**。

> おおかみ・おおせ・おおやけ・こおり（氷・郡）・こおろぎ・ほお（頬・朴）・ほおずき・ほのお・とお（十）・
> いきどおる（憤る）・おおう（覆う）・こおる（凍る）・しおおせる・とおる（通る）・とどこおる（滞る）・
> もよおす（催す）・いとおしい・おおい（多い）・おおきい（大きい）・とおい（遠い）・おおむね・おおよそ

これらが「う」を添えた長音（おうかみ・こうろぎ 等）で誤記された箇所を検出し、「お」表記への修正を提案する。

## 出典

現代仮名遣い（昭和61年内閣告示第1号）本文 第2の6

## 例

| 区分 | テキスト             | 期待                        |
| ---- | -------------------- | --------------------------- |
| 正例 | `おおかみが現れた。` | 指摘なし                    |
| 誤例 | `おうかみが現れた。` | `おうかみ` を `おおかみ` に修正 |
| 誤例 | `こうろぎが鳴く。`   | `こうろぎ` を `こおろぎ` に修正 |
| 誤例 | `おうきな声。`       | `おうき` を `おおき` に修正    |

## 偽陽性回避（高精度方針）

本ルールは告示の語例のうち、**誤記形が独立した別語と衝突しない語のみ**を収録する。これにより
小説本文での誤検出を避ける。

- **除外語（誤記形が別語と衝突）**:
  - `氷（こおり）` → 誤記形 `こうり` が `小売・行李・高利` と衝突。
  - `十（とお）` → `とう`（塔・問う・党 …）と衝突。
  - `頬（ほお）` → `ほう`（方・法 …）と衝突。
  - `仰せ（おおせ）` → `おうせ`（逢瀬）と衝突。
  - `多い（おおい）` → `おうい`（王位）と衝突。
  - `遠い（とおい）` → `とうい`（等位・糖衣・問い …）と衝突。
  - `通る（とおる）`・`凍る（こおる）`・`覆う（おおう）`・`催す（もよおす）` も誤記形が漢字音や別語と
    紛れやすいため、本ルールでは扱わない。
- **収録語**（誤記形が別語にならない語のみ）:
  - `おおかみ（狼）`・`おおやけ（公）`・`こおろぎ（蟋蟀）`・`ほのお（炎）`・`いとおしい（愛おしい）`・
    `おおむね（概ね）`・`おおよそ（凡そ）`・`ほおずき（鬼灯）`・`大きい（おおきい／大きな／大きく）`
  - 活用語 `憤る（いきどおる）`・`滞る（とどこおる）` は語幹＋直後の活用語尾（先読み）でのみ一致させる。
- `大きい` は `おうき＋（い／な／く／かっ）` に限定し、`王気` 等への巻き込みを防ぐ。

## 実装メモ

- `ctx.toolkit.regexReplace` を語ごとに適用し、活用語は先読み `(?=…)` で語尾を限定。最後に
  `ctx.toolkit.dedupe` で同一スパンの重複を除去。
- 対象語は内閣告示の列挙に基づく**有限リスト**。語を追加する際は、誤記形が別語と衝突しないこと
  （偽陽性テスト）を必ず確認する。
